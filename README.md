# 課題5 -API-

## アプリの使い方
**トップページ**  
トップページの商品画像をクリックすると詳細画面へ移動します。  
  
**商品画面**  
「Add to cart」ボタンをクリックするとカートに商品が入ります。  
    
**カート画面**  
右上のカートボタンを押すとカート画面に遷移します。  
「Check out」ボタンをクリックすると Stripe の支払い画面に遷移します。  
  
## ① 課題内容（どんな作品か）
Laravel と Stripe で作成するシャレオツな T シャツ EC ショップ
